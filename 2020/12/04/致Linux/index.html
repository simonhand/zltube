Linux的介绍以及常用的命令（上）

linux的介绍

1、简介：

  • 操作系统对与计算机来说就是它的灵魂所在，都说好看的皮囊千篇一律，有趣的灵魂万里挑一，linux对编程和项目部署都是非常nice的。
    • 对于操作系统的定义就是：直接运行在计算机上的系统软件， 它是控制硬件和支持软件运行的计算机程序。

linux与windows的区别

  • windows的文件目录

  • linux的文件目录

  • linux的目录更像像是一颗树

  • 所以这样结构其实很好理解绝对路径与相对路径

    绝对路径就是从根目录(/)这里出发，在win中就是从盘符出发的文件路径，绝对路径的好处就是基本上不会出错，但是往往路径很长，而且可移植性差。

相对路径则是要一个参照物，而这个参照就是代码文件所处在的当前目录

cd /home/zl/Desktop/1.txt # 这个就是典型的绝对路径打开桌面上的一个txt文件 
cd ../static/images # 这个就是相对路径的示例，..代表上一级目录

# 这里提示下~代表用户的主目录，/则是linux的根目录，
# cd . 是切换到当前目录

# cd - 是切换到上一次目录

#

查看目录的命令

命令 说明
ls 查看当前目录所拥有的文件或文件夹信息
tree 以树状方式显示目录信息
pwd 查看当前文件夹所在的路径
  • ls的选项命令
命令选项 说明
-l 以列表的形式表示
-h 以大小单位显示，默认时字节
-a 这里a意思all显示s所有的文件包括隐藏文件

  • 当然你觉得你的终端显示的太多东西时可选用clean来清楚一下

    常用操作文件命名

    命令 说明
    touch “文件名” 创建指定文件
    mkdir 目录名 创建目录(文件夹)
    rm 文件名或者目录名 删除指定文件或者目录
    rmdir 目录名 删除空目录 
    # 在删除带有文件的文件夹时需使用
rm -f # 递归的方式删除文件夹
# ps删库跑路的梗就是这个命令哦

    复制、移动文件及目录命令的使用

    命令 说明
    cp 拷贝文件、拷贝目录
    mv 移动文件、移动目录、重命名 
    cp "复制文件的路径" "复制到的路径"
mv "移动文件的路径" "移动到的路径"
mv "文件的旧命名" "新命名"
    • mkdair的命令选项
命令选项 说明
-p 创建依赖的文件夹
  • rm的命令选项
命令选项 说明
-i 交互式提示
-r 递归删除目录及其内容
-f 强制删除，忽略不存在的文件，无需提示
-d 删除空目录
  • rm的命令选项
命令选项 说明
-i 交互式提示
-r 递归拷贝目录及其内容
-v 显示拷贝后的路径描述
-a 保留文件的原有权限
  • mv 命令选项
命令选项 说明
-i 交互式提示
-v 显示移动后的路径描述
